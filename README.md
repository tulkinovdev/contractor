## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/tulkinovdev/contractor.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm run dev
```

## Legacy Version

The original project (HTML, CSS, JavaScript) is available in a separate branch:
